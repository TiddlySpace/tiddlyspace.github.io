So long, and thanks for all the tiddlers

The service at tiddlyspace.com has now ceased operation. It was created in 2010 by the Osmosoft team at BT, and generously operated for the community by them and their successors until December 9th 2016.

TiddlySpace has always been an open source project, and the code is available on GitHub so that anyone can host their own instance.

Please use the TiddlySpace mailing list if you have any questions.
